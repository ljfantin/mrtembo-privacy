Política de Privacidade - Mr Tembo

Última atualização: março 2026

Mr Tembo é um jogo gratuito desenvolvido pela Oli Dream Lab.

Dados que coletamos

Mr Tembo exibe anúncios através do Google AdMob. O AdMob pode coletar dados do dispositivo, como identificadores de publicidade, para exibir anúncios relevantes. Para mais informações consulte a política de privacidade do Google.

Também utilizamos o GameAnalytics para coletar dados anônimos de uso do jogo com o objetivo de melhorar a experiência de jogo. Para mais informações consulte a política de privacidade do GameAnalytics.

Dados que NÃO coletamos

  • Não coletamos nomes, e-mails ou dados pessoais
  • Não coletamos dados de localização
  • Não coletamos informações de pagamento

Crianças

Este jogo é destinado ao público em geral, incluindo crianças. Não coletamos intencionalmente informações pessoais de crianças menores de 13 anos.
